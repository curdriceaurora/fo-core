# TUI Tour

![TUI overview](../assets/tui-overview.svg)

This tour walks through the Textual-based terminal UI and its most important keybindings.

## Launch the TUI

```bash
file-organizer tui
```

## Global Navigation

- `1` Files
- `2` Organized
- `3` Analytics
- `4` Methodology
- `5` Audio
- `6` History
- `7` Settings
- `8` Copilot
- `Tab` cycle focus
- `?` help
- `q` quit

## Files View (Browse + Select)

- `j` / `k` move down/up
- `h` go to parent
- `l` expand/select
- `/` filter by extension
- `Space` toggle selection
- `Ctrl+A` select all
- `Ctrl+D` deselect all

## Organized View (Preview)

- `r` refresh preview
- `Enter` confirm (placeholder)
- `Esc` cancel

## Analytics View

- `r` refresh analytics

## Audio View

- `r` rescan audio
- `j` / `k` next/previous file

## History View

- `r` refresh history
- `u` undo last operation
- `y` redo last undone operation

## Methodology View

- `p` switch to PARA
- `j` switch to Johnny Decimal
- `n` switch to none
- `m` migrate (placeholder)

## Copilot View

- `Enter` submit message
- `Esc` clear input
