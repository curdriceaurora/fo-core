## Technical Details

### Python Version

- **Minimum**: Python 3.9+ (set project-wide in Phase 5)
- **Recommended**: Python 3.12 for development
- **pyproject.toml**: `requires-python = ">=3.9"` (already set)

### Dependencies (Already in pyproject.toml)

- **typer[all]>=0.12.0**: CLI framework with type hints and auto-completion
- **textual>=0.50.0**: Terminal UI framework for interactive interfaces (bumped from 0.48.0)
- **pyyaml>=6.0.0**: YAML parsing for configuration files
- **rich>=13.0.0**: Rich terminal output (transitive via typer[all])

### Setup Steps

1. Create Python 3.9+ virtual environment:
   ```bash
   cd file_organizer_v2
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install with dev dependencies:
   ```bash
   pip install -e ".[dev]"
   ```

3. Verify imports:
   ```bash
   python3 -c "import typer; import textual; import yaml; print('All Phase 2 deps OK')"
   ```

4. Run smoke tests:
   ```bash
   pytest tests/test_phase2_deps.py -v
   ```
