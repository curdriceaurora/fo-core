# Task: Add Deduplication and Intelligence CLI Commands

## Description
Extend the Typer CLI with commands for Phase 4 deduplication (scan, review, resolve duplicates), intelligence features (view suggestions, apply preferences, manage profiles), and analytics (storage stats, organization quality metrics).

## Acceptance Criteria
- [ ] `file-organizer dedupe scan <path>` - Scan for duplicates
- [ ] `file-organizer dedupe resolve` - Interactive duplicate resolution
- [ ] `file-organizer suggest <path>` - Show PARA/organization suggestions
- [ ] `file-organizer profile list/export/import` - Manage preference profiles
- [ ] `file-organizer analytics summary` - Show organization statistics
- [ ] All commands support --dry-run and --json output

## Technical Details
- Integrate with services/deduplication/ (detector, image_dedup, document_dedup)
- Integrate with services/intelligence/ (preference_tracker, profile_manager)
- Integrate with services/analytics/ (analytics_service)
- Build on existing cli/ module structure
