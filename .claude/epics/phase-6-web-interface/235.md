# Task 007: Implement File Browser with Thumbnails

## Description

Build a comprehensive file browser UI that allows users to navigate directories, view file thumbnails (images, PDFs, videos), preview files in a modal, upload files via drag-and-drop, and perform bulk file operations. The interface should be intuitive, fast, and responsive.

## Acceptance Criteria

- [ ] Directory tree navigation (expandable/collapsible folders)
- [ ] File list view with grid and list toggle
- [ ] Thumbnails for images, PDFs, and video files
- [ ] File preview modal with details (name, size, type, date, metadata)
- [ ] Drag-and-drop file upload to current directory
- [ ] Bulk file selection with checkboxes
- [ ] Context menu for file operations (rename, move, delete)
- [ ] Breadcrumb navigation
- [ ] Search/filter files by name or type
- [ ] Sorting (name, date, size, type)
- [ ] Pagination or infinite scroll for large directories
- [ ] Loading states and error handling
- [ ] Keyboard navigation support

## Technical Details

### Components

#### 1. Directory Tree (Sidebar)
- Collapsible folder structure
- HTMX for lazy loading subdirectories
- Active folder highlighting

### Thumbnail Generation

Use existing metadata extraction for thumbnails:
- **Images**: Resize to 200x200px, cache
- **PDFs**: Extract first page, resize
- **Videos**: Extract frame at 10% duration
- **Audio**: Use default icon with waveform preview
- **Documents**: Use file type icon
