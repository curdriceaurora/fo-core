# Task 001: Setup FastAPI Backend Infrastructure

## Description

Create the foundational FastAPI application structure for the File Organizer web interface. This includes setting up the project scaffolding with proper separation of concerns (routers, dependencies, middleware), configuring CORS for frontend communication, implementing comprehensive logging, adding error handlers, and setting up OpenAPI/Swagger documentation.

The backend will serve as the HTTP/WebSocket gateway to all File Organizer functionality, making CLI features accessible via REST API.

**3. Middleware (`middleware.py`)**
```python
from fastapi import Request
import time
import logging

async def log_requests_middleware(request: Request, call_next):
    start_time = time.time()
    response = await call_next(request)
    duration = time.time() - start_time
    logging.info(f"{request.method} {request.url.path} - {response.status_code} - {duration:.3f}s")
    return response

def setup_middleware(app):
    app.middleware("http")(log_requests_middleware)
```

**4. Exception Handlers (`exceptions.py`)**
```python
from fastapi import Request, status
from fastapi.responses import JSONResponse

class FileOrganizerException(Exception):
    def __init__(self, message: str, status_code: int = 500):
        self.message = message
        self.status_code = status_code

async def file_organizer_exception_handler(request: Request, exc: FileOrganizerException):
    return JSONResponse(
        status_code=exc.status_code,
        content={"error": exc.message}
    )

def setup_exception_handlers(app):
    app.add_exception_handler(FileOrganizerException, file_organizer_exception_handler)
```

def setup_logging():
    logging.basicConfig(
        level=logging.INFO,
        format="%(asctime)s - %(name)s - %(levelname)s - %(message)s",
