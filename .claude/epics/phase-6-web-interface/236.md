# Task 008: Create Organization Dashboard

## Description

Build a comprehensive organization dashboard that allows users to scan directories, preview organization plans, execute organization operations, track progress in real-time, view organization history and logs, and analyze statistics. The dashboard should provide clear visibility into the organization process and results.

## Acceptance Criteria

- [x] Directory scan interface with path selection
- [x] Methodology selector (Johnny Decimal, PARA, etc.)
- [x] Organization plan preview with file movements
- [x] Approve/reject/modify plan interface
- [x] Execute organization with progress tracking
- [x] Real-time progress updates (WebSocket or SSE)
- [x] Organization history/logs viewer
- [x] Statistics dashboard (files organized, rules applied, etc.)
- [x] Error handling and rollback capability
- [x] Dry-run mode toggle
- [x] Schedule organization jobs
- [x] Export organization reports

## Technical Details

### Workflow Steps

1. **Select Directory** → 2. **Choose Methodology** → 3. **Scan & Generate Plan** → 4. **Review Plan** → 5. **Execute** → 6. **View Results**

#### 5. Real-Time Updates

Use Server-Sent Events (SSE) for progress updates:

```javascript
const eventSource = new EventSource('/api/organize/progress/123');

eventSource.addEventListener('progress', (e) => {
  const data = JSON.parse(e.data);
  updateProgressBar(data.percentage);
  updateStats(data.processed, data.total);
});

eventSource.addEventListener('log', (e) => {
  const data = JSON.parse(e.data);
  appendLogEntry(data.type, data.message);
});

eventSource.addEventListener('complete', (e) => {
  showCompletionSummary(JSON.parse(e.data));
  eventSource.close();
});
```

### WebSocket Integration (Alternative to SSE)

```python
from fastapi import WebSocket

@router.websocket("/ws/organize/{job_id}")
async def websocket_progress(websocket: WebSocket, job_id: str):
    await websocket.accept()
    try:
        while True:
            progress = await get_job_progress(job_id)
            await websocket.send_json(progress)
            if progress['status'] == 'complete':
                break
            await asyncio.sleep(1)
    except WebSocketDisconnect:
        pass
```
