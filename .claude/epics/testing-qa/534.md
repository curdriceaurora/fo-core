# Issue #534: Add missing pytest markers to API auth test files

## Description

Three API auth test files lacked `@pytest.mark.unit` markers, preventing selective
test execution by category.

## Files Modified

- `tests/api/test_auth.py` — 7 test classes marked
- `tests/api/test_auth_rate_limit.py` — 3 test classes marked (+ added `import pytest`)
- `tests/api/test_auth_store.py` — 2 test classes marked (+ added `import pytest`)

## Resolution

Added `@pytest.mark.unit` to all 12 test classes across the 3 files. All 61 tests
pass and are correctly selected via `pytest -m unit`.

Resolved in PR #536.
