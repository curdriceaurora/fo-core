# Task: Add Timestamp Validation to Pre-Commit Script

## Description

Add 4 new datetime validation checks to `.claude/scripts/pre-commit-validation.sh` that catch naive datetime patterns on staged files before they can be committed. This creates a defensive gate that prevents new timestamp bugs from reaching main.

## Acceptance Criteria

- [ ] Check 1: Detect `datetime.now()` without UTC in staged Python files — fail
- [ ] Check 2: Detect deprecated `datetime.utcnow()` in staged Python files — fail
- [ ] Check 3: Detect `fromtimestamp(` without `tz=` in staged Python files — warn
- [ ] Check 4: Detect `.isoformat()` followed by `+ "Z"` or `+"Z"` pattern — fail
- [ ] All 4 checks integrated into the existing pre-commit validation flow
- [ ] Each check has clear error message with fix instructions
- [ ] Existing 13 checks continue to work unchanged

## Technical Details

File: `.claude/scripts/pre-commit-validation.sh` (~422 lines, 13 existing checks)

Add a new section after the existing checks. Implementation approach:

```bash
# Check 14: Naive datetime.now() detection

echo "Check 14: Naive datetime patterns..."
STAGED_PY_FILES=$(git diff --cached --name-only --diff-filter=ACM | grep '\.py$')
if [ -n "$STAGED_PY_FILES" ]; then
  # Check for datetime.now() without UTC/timezone
  NAIVE_NOW=$(echo "$STAGED_PY_FILES" | xargs grep -n 'datetime\.now()' 2>/dev/null | grep -v 'now(UTC\|now(timezone\|now(tz=' || true)
  if [ -n "$NAIVE_NOW" ]; then
    echo "❌ Found naive datetime.now() — use datetime.now(UTC) instead:"
    echo "$NAIVE_NOW"
    FAILED=1
  fi

  # Check for deprecated utcnow()
  UTCNOW=$(echo "$STAGED_PY_FILES" | xargs grep -n 'datetime\.utcnow()' 2>/dev/null || true)
  if [ -n "$UTCNOW" ]; then
    echo "❌ Found deprecated datetime.utcnow() — use datetime.now(UTC) instead:"
    echo "$UTCNOW"
    FAILED=1
  fi

  # Check for bare fromtimestamp()
  BARE_TS=$(echo "$STAGED_PY_FILES" | xargs grep -n 'fromtimestamp(' 2>/dev/null | grep -v 'tz=' || true)
  if [ -n "$BARE_TS" ]; then
    echo "⚠️ Found fromtimestamp() without tz= — add tz=UTC parameter:"
    echo "$BARE_TS"
  fi

  # Check for isoformat()+"Z" trap
  ISO_TRAP=$(echo "$STAGED_PY_FILES" | xargs grep -n 'isoformat().*+.*"Z"' 2>/dev/null || true)
  if [ -n "$ISO_TRAP" ]; then
    echo "❌ Found isoformat()+\"Z\" trap — use .isoformat().replace(\"+00:00\", \"Z\") instead:"
    echo "$ISO_TRAP"
    FAILED=1
  fi
fi
```
