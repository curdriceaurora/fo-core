# Task: Fix 27 Naive datetime.now() Calls

## Description

Replace all 27 instances of `datetime.now()` (without timezone) with `datetime.now(UTC)` using `from datetime import UTC`. This eliminates DTZ005 violations and prevents naive/aware datetime comparison crashes.

- [ ] All existing tests pass after changes
