# Task: Full System Tray Menu

## Description

Extend the basic system tray (Task 007) with full menu: Organize Now, Recent Activity, Pause/Resume Daemon, Settings, About, Quit. Each item calls the appropriate REST API endpoint.

## Acceptance Criteria

- [ ] "Organize Now" triggers POST `/api/v1/organize`
- [ ] "Recent Activity" shows submenu with last 5 operations from `/api/v1/history/recent`
- [ ] "Pause/Resume" toggles daemon via `/api/v1/daemon/toggle`
- [ ] "Settings" opens main window to settings page
- [ ] "About" shows version info dialog
- [ ] Tray icon changes based on daemon state (active/paused)
- [ ] Integration tests validate tray menu -> API mapping

## Technical Details
