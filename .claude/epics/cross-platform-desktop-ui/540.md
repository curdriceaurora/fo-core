# Task: Implement Splash Screen

## Description

Show a splash screen while the Python sidecar is starting up. Transition to the main Web UI once the health endpoint reports ready.

## Acceptance Criteria

- [ ] Splash screen HTML/CSS with app logo and loading indicator
- [ ] Splash screen shows during sidecar startup (health polling phase)
- [ ] Smooth transition to main Web UI once backend is ready
- [ ] Error state shown if sidecar fails to start after timeout
- [ ] Splash works on macOS, Windows, Linux

## Technical Details

- Create `desktop/src/splash.html` with inline CSS (no external deps)
- Use Tauri window API to show splash, hide when ready
- Listen for sidecar "ready" event to trigger navigation to `http://localhost:{PORT}/ui/`
- Error state: show retry button and log location
