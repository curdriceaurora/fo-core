# Task: Implement Basic System Tray

## Description

Add basic system tray icon with minimal menu (Open Window, Quit) using `tauri-plugin-tray`. This is the foundation for the full tray menu in Phase 4.

## Acceptance Criteria

- [ ] System tray icon appears when app runs
- [ ] Tray menu has "Open" and "Quit" items
- [ ] "Open" shows/focuses main window
- [ ] "Quit" triggers clean shutdown (sidecar + app)
- [ ] Tray icon uses app icon (.ico on Windows, .png on Linux, template image on macOS)
- [ ] Works on macOS, Windows, Linux

## Technical Details

- Create `desktop/src/tray.rs`
- Use `tauri-plugin-tray` for cross-platform tray
- Connect "Quit" to sidecar shutdown flow
- App icon from `desktop/icons/` (will be generated in Task 010)
- Use placeholder icon until Task 010 completes
