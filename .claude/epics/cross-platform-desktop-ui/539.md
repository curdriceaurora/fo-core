# Task: Implement Daemon Manager - Linux

## Description

Create Rust module for managing the file organizer daemon via systemd user service. Handles install/uninstall of unit file, start/stop/status via systemctl --user.

## Acceptance Criteria

- [ ] `desktop/src/daemon/linux.rs` implements DaemonManager trait
- [ ] Generates valid systemd unit file with correct sections
- [ ] Installs unit to `~/.config/systemd/user/`
- [ ] Start/stop/status/enable via `systemctl --user` commands
- [ ] Auto-launch on login via systemd enable
- [ ] Rust unit tests with unit file generation

## Technical Details

- Create `desktop/src/daemon/linux.rs` implementing DaemonManager trait
- systemd unit sections: [Unit], [Service], [Install]
- Service type: simple, with restart on failure
- Use `std::process::Command` for `systemctl` calls
- Create `tests/daemon/test_native_daemon.py` (Linux section)
