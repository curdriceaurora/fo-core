# Task: Generate App Icons

## Description
Generate platform-specific app icons from the existing logo.png: .icns for macOS, .ico for Windows, and multi-resolution PNGs for Linux.

## Acceptance Criteria
- [ ] macOS: `.icns` file with resolutions 16, 32, 128, 256, 512, 1024
- [ ] Windows: `.ico` file with resolutions 16, 32, 48, 256
- [ ] Linux: PNG files at 48, 128, 256 resolutions
- [ ] macOS tray template image (white on transparent, @1x and @2x)
- [ ] All icons placed in `desktop/icons/`
- [ ] CI test validates icon formats and resolutions

## Technical Details
- Source: existing `logo.png` or generate from project branding
- Use `iconutil` (macOS) or `png2icns` for .icns generation
- Use ImageMagick or Pillow for .ico and multi-res PNG
- Create `desktop/icons/` directory structure
- Create `tests/ci/test_icons.py` for validation
- Consider creating a script `scripts/generate_icons.sh`

## Dependencies
- [ ] None

- Size: S

- Parallel: true

## Definition of Done
- [ ] All icon formats generated correctly
- [ ] Icon tests passing
- [ ] Icons look good at all resolutions
