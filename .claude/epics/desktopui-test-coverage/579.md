# Task: Docstring Coverage via Interrogate

## Description
Add missing docstrings across the codebase to reach the 90% threshold configured in pyproject.toml. Run `interrogate` to identify gaps and systematically fill them.

## Scope (MECE Boundary)
**Includes**: Module-level, class-level, and function-level docstrings in all `.py` files under `src/file_organizer/`.
**Excludes**: Test file docstrings (test files covered by their respective tasks), inline code comments, type annotations.
