# Reference & Support

## Supported File Types (Total: 43)
- **Documents**: PDF, DOCX, TXT, MD, EPUB, PPTX, XLSX...
- **Images**: JPG, PNG, GIF, BMP, TIFF...
- **Video**: MP4, AVI, MKV, MOV, WMV.
- **Archive**: ZIP, TAR, 7Z, RAR.
- **Scientific**: HDF5, NetCDF, MAT.

## Performance Benchmarks
- **Text (<1MB)**: 2-5s (Qwen 3B)
- **Image**: 3-8s (Qwen VL)
- **Memory**: Requires ~8GB RAM minimum.

## Dependencies
- `ollama>=0.1.0`
- `typer[all]>=0.12.0`
- `sqlalchemy>=2.0.0`
