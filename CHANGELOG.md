## [2.0.0] - 2026-02-07

### Added
- AI-powered file organization with local LLMs (Ollama)
- Privacy-first architecture with no cloud data transmission
- PARA methodology support for file categorization
- CLI interface with `file-organizer` and `fo` commands
- Audio file processing with Whisper integration
- Video scene detection support
- Image duplicate detection
- Archive file format support (7z, RAR)
- Scientific file format support (HDF5, NetCDF, MATLAB)
- CAD file format support (DXF/DWG)
- Event-driven architecture for file processing
- Undo/redo support for file operations
- File watcher for automated organization
- FastAPI-based web interface
- SQLAlchemy-based history tracking
- Comprehensive test suite

### Changed
- Complete rewrite from v1.0 architecture
- Upgraded to Python 3.9+ compatibility
- Switched to Typer for CLI framework
- Adopted pydantic v2 for configuration
